Welcome to my Python Reference Page!!


I created this page as my final project for GSP 318 - Geospatial Programming at Cal Poy Humboldt.


This class used Python to accomplish geospatial tasks and I wanted to create a place to store code snippets and reference material from class and other websites.


The bulk of the material can be accessed from the Python Reference link at the top of the page or by clicking specific topics from the table of contents on the left. Python Samples will take you to some longer blocks of Python code and Other Sites will provide links to other Python reference and documentation.
