Basic Math Operations

                
print(1 + 2)   # Addition
print(1 - 2)   # Subtraction
print(10 * 5)  # Multiplication
print(10 / 5)  # Division
print(10 ** 5) # Exponent
print(10 // 5) # Integer division
print(10 % 5)  # Modulus/remainder
print(5 + 9.55 * (5 - 2) / 5 + 2 ** 2) # Multiple operations (follow order of operations)

Rules for Variable Names

- No spaces

- Letters, numbers, and underscore (_) only

- Cannot begin with number

- Case sensitive



Concatenate Strings

                
FirstName = "John"
LastName = "Doe"
print(FirstName + LastName)
print(FirstName + " " + LastName) # With space in between names
print(FirstName * 5) # Print name 5 times

Comparison Operators

- Equal to: ==

- Not equal to: !=

- Lesser than: <

- Greater than: >

- Lesser than or equal: <=

- Greater than or equal: >=

                
print(42 == 42) # return True
print(42 == "42") # return False

print(2 != 5) # return True
print((2 != 2) # return False

print(5 < 10) # return True
print(5 <= 5.0) # return False

Logical Operators

- and

- or

- not

                    
print((4 < 6) and (6 < 7))
print((4 < 6) and (8 < 7))
